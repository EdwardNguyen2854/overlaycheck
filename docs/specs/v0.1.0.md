# OverlayCheck v0.1.0 — Release Spec

## Theme

Local-first PDF overlay comparison for CAD and document revision workflows. No server, no telemetry, no accounts. Files stay in the browser.

## Scope

### In

- PDF A (base) + PDF B (overlay) file upload from device
- Overlay compare modes: overlay, A-only, B-only, blink
- Momentary "Show A / Show B" press-and-hold controls
- Configurable blink rate (Slow/Standard/Fast) with pause
- Mouse wheel zoom to cursor
- Drag pan (Shift + drag)
- Manual PDF B alignment: X/Y offset, scale, rotation
- Drag PDF B directly in viewer
- Nudge buttons and keyboard nudges (1px, 10px)
- Light and dark themes
- Responsive below 1080px
- Browser localStorage persistence for alignment and review data
- Keyboard-accessible focus, labels, live announcements
- `prefers-reduced-motion` respected

### Out

- Backend server
- Cloud storage or file upload server
- User accounts / authentication
- Telemetry or analytics
- Pixel diff mode
- Side-by-side mode
- Auto-alignment
- Shared annotations

## Milestone

Target: initial public release
